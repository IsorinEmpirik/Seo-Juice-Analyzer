Comment générer les embeddings avec Gemini
Prérequis : Une clé API Google Gemini (gratuite) Obtenir une clé API
Configurer Screaming Frog
  1. Ouvrez Screaming Frog et allez dans Configuration → API Access → Gemini
  2. Entrez votre clé API Gemini
  3. Dans la section "Custom Extraction", cochez "Extract embeddings from page content"
  4. Lancez votre crawl normalement
Exporter le fichier
  1. Une fois le crawl terminé, allez dans Bulk Export → All → Internal
  2. Sélectionnez le format CSV
  3. Le fichier doit contenir la colonne "Extract embeddings from page content"
Important : Les embeddings sont des vecteurs numériques représentant le contenu sémantique de chaque page. Ils permettent de calculer la similarité entre vos pages.
