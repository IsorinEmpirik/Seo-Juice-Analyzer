Comment generer les embeddings
Compatible Gemini et OpenAI. L'outil detecte automatiquement le fournisseur et le format des embeddings.
Prerequis : Obtenir une cle API Gemini
1. Configurer Screaming Frog
  1. Configuration > API Access > Gemini
  2. Entrez votre cle API
  3. Cochez "Extract embeddings from page content"
  4. Lancez le crawl
2. Exporter
  1. Bulk Export > All > Internal > CSV
Prerequis : Obtenir une cle API OpenAI
1. Configurer Screaming Frog
  1. Configuration > API Access > OpenAI
  2. Entrez votre cle API
  3. Cochez "Extract embeddings from page content"
  4. Lancez le crawl
2. Exporter
  1. Bulk Export > All > Internal > CSV
CSV personnalise : Vous pouvez aussi fournir un CSV avec 2 colonnes : une colonne URL et une colonne contenant les vecteurs (nombres separes par des virgules). L'outil detecte automatiquement le format.
Format attendu

CSV avec au minimum 2 colonnes :

  • URL : colonne nommee "Adresse", "URL" ou "Address"
  • Embedding : colonne contenant les vecteurs de nombres (detectee automatiquement)
